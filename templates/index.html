Inclusive STL

Inclusive STL is an project to convert text to braille and export it as a STL file

Please note: In Brazilian alphabet, the braille to text conversion has problems with some characters. For example, the letter "&" has the same braille as the letter "ç", for this reason, the conversion will be made to the letter "ç".

Legend of Advanced Options:

  • Unique Plate:

    If checked, the phrase will be in a single plate.

  • Unique Width:

    If checked, the width of all plates will be the same.

  • Rounded:

    If checked, the edges will be rounded.

  • Dots Radius:

    The radius of the braille dots.

  • Spacing:

    The spacing between the braille symbols.

  • Plate Thickness:

    The thickness of the plates.

  • Kerning (Based on Spacing):

    The kerning of the text.

  • Text Alignment:

    The alignment of the text.

  • Dots Subdivisions:

    The subdivisions of the braille dots. By server limitations, the maximum value is 4 and the minimum is 1.
